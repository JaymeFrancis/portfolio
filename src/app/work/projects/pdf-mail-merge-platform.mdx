## Overview

A modern web application for **PDF mail merging** built with the **RILT stack** (React, InertiaJS, Laravel, and TailwindCSS).  
The platform allows users to **create, manage, and process PDF documents** with dynamic data, including **team collaboration** and **subscription management** features.  
I led the development of this project and delivered a fully functional **MVP in just one and a half weeks**.

---

## Key Features

#### PDF Processing

- Upload and manage PDF templates with **drag-and-drop functionality**
- Add **dynamic variables** to PDFs for mail merge operations
- **Batch processing** support for multiple documents
- Custom filename generation for processed files

#### Team Collaboration

- Team creation and management system
- **Role-based access control** for documents and features
- Real-time collaborative document editing
- Team invitation and member management

#### Document Management

- Secure file storage and organization
- Version control for documents
- Document sharing with permission levels
- **Audit trail** of document changes

#### Data Integration

- Excel/CSV data import for mail merge
- Variable mapping interface
- Batch processing status tracking
- Data validation and error handling

#### User Management

- Secure authentication with **Laravel Jetstream**
- Profile management with 2FA and email verification
- **Role-based access** and granular permissions

#### Subscription & Billing

- Tiered subscription plans
- Secure payment processing via **Stripe**
- Billing history and invoice management
- Team-based billing system

---

## Technologies Used

- **Backend:** Laravel 11 (PHP)
- **Frontend:** React with InertiaJS
- **Styling:** TailwindCSS with DaisyUI and Shadcn/UI components
- **Database:** MySQL / PostgreSQL
- **Authentication:** Laravel Jetstream with Sanctum
- **Payment Processing:** Stripe Integration
- **PDF Processing:** TCPDF, FPDI, and other PHP PDF libraries
- **Build Tools:** Vite.js
- **Testing:** PHPUnit for backend testing

---

## Architecture

- **Modular Design:** Separated modules for PDF processing, user management, and team collaboration
- **API-First Architecture:** Built with a robust API layer for future extensibility
- **Responsive UI:** Fully responsive design using TailwindCSS
- **Component Library:** Reusable UI components via Shadcn/UI and DaisyUI
